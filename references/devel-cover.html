Devel::Cover

Tipo: Repositório Open Source
Autor(es): Paul Johnson
Publicado: 2001 (atualizado continuamente)
Acessado: 2026-06-26

Relevância​

Devel::Cover é o módulo padrão para métricas de cobertura de código em Perl 5. Gera relatórios de cobertura de declarações, branches, condições, sub-rotinas e POD. Integra com sistemas de CI e ferramentas como Coveralls. Referência para a ADR de estratégia de testes — cobertura de código é um requisito do stack para serviços em produção.
