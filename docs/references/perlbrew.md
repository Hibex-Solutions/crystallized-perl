## Relevância
Perlbrew é o gerenciador de versões de Perl para ambientes Unix/Linux/macOS. Permite
instalar e alternar entre múltiplas versões do Perl sem afetar o Perl do sistema
operacional. Referência para a seção de configuração do ambiente de desenvolvimento
local no stack — componente essencial para garantir a versão mínima de Perl declarada
no projeto.
