# [github/gitignore — Coleção de templates .gitignore](https://github.com/github/gitignore)

**Tipo**: Repositório Open Source  
**Autor(es)**: GitHub, Inc. e colaboradores da comunidade  
**Publicado**: 2013 (atualizado continuamente)  
**Acessado**: 2026-06-27

## Relevância
Repositório oficial do GitHub com templates de `.gitignore` mantidos pela comunidade
para dezenas de linguagens e ambientes, incluindo Perl, Docker, JetBrains e editores.
No stack Crystallized Perl, o template `Perl.gitignore` serve de base para o arquivo
`.gitignore` padrão de todo projeto — adaptado para excluir artefatos do Carton
(`local/`), saídas de build e arquivos de ambiente local (`.env`).
