# [Mojolicious — Perl real-time web framework](https://mojolicious.org/)

**Tipo**: Documentação Oficial

**Acessado**: 2026-06-26

## Relevância
Mojolicious é o framework web Perl moderno mais ativo e abrangente. Oferece servidor
HTTP assíncrono integrado, suporte a WebSocket, roteamento, templates, cliente HTTP
e testes — tudo sem dependências externas além do core. Candidato principal para a ADR
de framework web do stack.
