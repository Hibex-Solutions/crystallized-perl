## Relevância
Tutorial oficial de programação orientada a objetos em Perl 5 moderno. Cobre o sistema
de OO nativo do Perl e recomenda o uso de sistemas de alto nível como Moose e Moo.
Referência central para ADRs sobre modelagem de domínio, organização de classes e
escolha de sistema de OO no stack.
