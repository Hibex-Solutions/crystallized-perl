## Relevância
CPAN é o repositório central de módulos Perl — equivalente ao npm (JavaScript) ou PyPI
(Python). Com mais de 200.000 módulos disponíveis, é a principal fonte de dependências
do ecossistema. Referência obrigatória para qualquer ADR ou guia que trate de
gerenciamento de dependências, instalação de módulos e distribuição de código Perl.
