# [Modern Perl: a Perl Tutorial (4ª edição)](http://modernperlbooks.com/)

**Tipo**: Livro  
**Autor(es)**: brian d foy  
**Publicado**: 2016  
**Acessado**: 2026-06-26

## Relevância
Referência mais abrangente e atualizada de práticas modernas em Perl 5. Cobre Moose/Moo
para OO, gerenciamento de módulos via CPAN, testes com Test::More, expressões regulares,
referências e estruturas de dados complexas — tudo com o padrão de qualidade que
diferencia Perl moderno do Perl legado.

A identidade visual laranja do livro inspirou diretamente a cor de mosaico laranja
(`#F97316`) da paleta do projeto (ADR-003), como aceno explícito a esta obra e à
cultura "Modern Perl" que o projeto abraça.
