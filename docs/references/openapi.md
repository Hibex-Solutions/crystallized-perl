## Relevância
A OpenAPI Specification (anteriormente Swagger) é o padrão aberto para descrição de APIs
HTTP. Define contratos de API em formato YAML ou JSON, cobrindo rotas, parâmetros,
esquemas de requisição/resposta e esquemas de segurança (incluindo Bearer JWT). No stack
Crystallized Perl, a especificação OpenAPI v3 é usada com o plugin
`Mojolicious::Plugin::OpenAPI` para validação automática de entrada e saída nas rotas da
API — eliminando código de validação manual e garantindo que a implementação sempre
corresponda ao contrato publicado.
