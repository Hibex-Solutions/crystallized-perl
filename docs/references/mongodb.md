## Relevância
MongoDB é a plataforma de banco de dados de documentos mais utilizada no mercado.
Modelo de dados flexível (BSON/JSON), escalabilidade horizontal nativa e ecossistema
amplo de ferramentas. Candidato para a ADR de banco de dados de documentos, avaliado
em conjunto com o DocumentDB open source como alternativa de licença aberta.
