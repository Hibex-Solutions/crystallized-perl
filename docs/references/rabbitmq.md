**Publicado**: 2007 (atualizado continuamente)  
**Acessado**: 2026-06-26

## Relevância
RabbitMQ é o message broker open source mais amplamente adotado. Implementa AMQP 0-9-1,
suporta múltiplos protocolos (MQTT, STOMP) e oferece roteamento flexível via exchanges
e filas. Candidato para a ADR de fila de mensagens — componente central para o
processamento assíncrono e workers em background definidos no escopo do stack.
