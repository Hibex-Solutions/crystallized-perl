## Relevância
Coleção de tutoriais colaborativos da maior comunidade de discussão Perl na web. Cobre
técnicas específicas, padrões de design e soluções para problemas comuns que muitas
vezes não estão na documentação oficial. Referência secundária para guias práticos
e para casos de uso avançados não cobertos pelo perldoc ou pelo Modern Perl.
