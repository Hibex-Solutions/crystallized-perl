# [Perl.org](https://www.perl.org)

**Tipo**: Portal Oficial  
**Autor(es)**: The Perl Foundation e comunidade Perl  
**Publicado**: 1994 (atualizado continuamente)  
**Acessado**: 2026-06-26

## Relevância
Portal oficial da linguagem Perl. Ponto de entrada central para downloads de versões
estáveis, links para documentação canônica, anúncios da comunidade e eventos. Confirma
o Perl como linguagem ativa e mantida — fundamento para qualquer decisão arquitetural
que justifique o uso de Perl em produção moderna.
