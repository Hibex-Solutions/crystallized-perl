## Relevância
Docker é a plataforma de containerização de referência. Todo serviço do stack roda em
contêiner — isso é um princípio não-negociável do projeto (cloud-native first). Referência
para a ADR de containerização e para os guias de configuração de ambiente de
desenvolvimento e produção.
