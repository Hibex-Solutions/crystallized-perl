# [berrybrew — Gerenciador Perl para Windows](https://github.com/dnmfarrell/berrybrew)

**Tipo**: Repositório Open Source  
**Autor(es)**: dnmfarrell e colaboradores  
**Publicado**: 2016 (atualizado continuamente)  
**Acessado**: 2026-06-26

## Relevância
Equivalente do Perlbrew para ambientes Windows. Permite instalar e alternar entre
múltiplas versões do Strawberry Perl sem conflitos com o sistema. Referência para a
seção de configuração do ambiente de desenvolvimento local no stack para usuários
Windows — garantindo que o projeto seja acessível em multiplataforma desde o início.

Documentação de uso: [berrybrew Usage Guide](https://github.com/dnmfarrell/berrybrew/blob/master/doc/berrybrew.md)
