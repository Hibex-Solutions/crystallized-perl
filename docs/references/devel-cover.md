## Relevância
Devel::Cover é o módulo padrão para métricas de cobertura de código em Perl 5. Gera
relatórios de cobertura de declarações, branches, condições, sub-rotinas e POD.
Integra com sistemas de CI e ferramentas como Coveralls. Referência para a ADR de
estratégia de testes — cobertura de código é um requisito do stack para serviços
em produção.
