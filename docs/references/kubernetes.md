## Relevância
Kubernetes é o orquestrador de contêineres padrão da indústria para ambientes
cloud-native. Gerencia deploy, escalabilidade e resiliência de serviços containerizados.
Referência para a ADR de orquestração do stack — a escolha entre Kubernetes e
alternativas mais simples é uma das decisões arquiteturais centrais do projeto.
