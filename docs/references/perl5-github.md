**Tipo**: Repositório Open Source  
**Autor(es)**: The Perl 5 Porters  
**Publicado**: 1994 (desenvolvimento ativo)  
**Acessado**: 2026-06-26

## Relevância
Repositório do interpretador Perl 5 no GitHub. Confirma que a linguagem é desenvolvida
ativamente em open source com releases regulares. Permite consultar changelogs de versões
específicas, o que fundamenta ADRs que declaram versão mínima suportada do Perl.

## Referenciada em
- [ADR-005](../adrs/ADR-005-gerenciamento-de-dependencias.md) — fundamenta a declaração da versão mínima do Perl (`requires 'perl', '5.038'` no cpanfile)
